Eclipse 4.6.3安装配置

发表于 2017-06-27 | 分类于 编程相关 ， IDE ， Eclipse | 阅读次数: |
字数统计: 1,668 | 阅读时长 ≈ 8

背景

之前一直使用 MyEclipse，当项目工程代码多时，实在是卡成狗，特此撰文以记之！

准备工作

系统环境：win10 64位，此处列出本人开发过程会用到的相关插件配置，大家根据自己所需选择性安装。

  1. 本地请先安装好 JDK， 安装配置。
  2. 最新 Eclipse 安装程序，下载地址。
  3. ant 安装程序，下载地址。
  4. maven 安装程序，下载地址。

安装步骤

一、安装配置 JDK

本文安装的 JDK 版本为1.7.0_79。

二、安装配置 ant

  1. 将下载的 *.zip 格式的安装包，找个地方解压后即可使用，比如 “D:\Program Files\Java\apache-ant-1.8.4”。
  2. 配置环境变量，在 path 后面添加ant的bin目录路径，比如 “XX;D:\Program Files\Java\apache-ant-1.8.4\bin”。

    三、安装配置 maven

  3. 将下载的 *.zip 格式的安装包，找个地方解压，比如 “D:\Program Files\Java\apache-maven-3.0.4”。
  4. 配置环境变量，添加系统变量 “M2_HOME”，值为 “D:\Program Files\Java\apache-maven-3.0.4”。
  5. Maven 仓库配置：

四、安装配置 eclipse

安装过程：
打开安装程序，会弹出一个选择所需安装的IDE列表，此处选择第一个“ Eclipse IDE For Java Developers” （或者第二个“ Eclipse IDE For JavaEE Developers” ），然后一直点下一步，安装过程中还会继续联网下载相关插件，如果出现报错提示无法连接之类的，那么你需要一个翻墙软件，先翻墙下。

Windows\Preferences 配置

  1. General \ Appearance \ Colors and Fonts
    • 点击“Basic\Text Font”，点击“Edit”按钮。
    • 在弹出的对话框中，根据需要修改字体大小，点击“确定”按钮。

  2. General\ContentTypes\Text\Java Properties File
    设置Default encoding，把下面的ISO-8859-1改为UTF-8或者GBK（推荐UTF-8），然后update。

  3. General \ Editors \ Text Editors \ Spelling（推荐）

去掉“Enable spell checking”的勾选框。

  1. General \ Workspace

    将工作空间文本文件默认编码设置为UTF-8。

  2. Ant \ Editor \ Formatter

    设置ant文件缩进为2。

  3. Ant \ Runtime

  4. Java \ Compiler \ Errors/Warnings

    1. 点击菜单”Window | References…”进入”References”对话框，选择”Java|Complier|Errors/Warnings”。
    2. 请按照以下表格中“设置值”一列的要求设置，左边的Eclipse默认值供参考。
    3. 和Eclipse默认值不同的设置用 加粗 标明。

8.　Java \ Installed JREs
确认使用安装的JDK，并配置JVM参数为：-Xms256m -Xmx1024m -XX:PermSize=256M -XX:MaxPermSize=256M；同时修改Java\Compiler中的Compile Level为1.7.

9.　Validation
关闭各种文件的校验，点击”Disable All”->”Apply”。

10.　XML \XML Files\Editors
XML文件缩进为2。

11.　相关插件的安装
eclipse初次安装时，选择”Eclipse IDE for JavaEE developers “的童鞋，可以少去部分插件的安装，此处介绍出本人用到的插件安装过程。

Maven\Installations
使用安装的Maven:

安装maven使用相关插件

  • 安装gef 插件，使用eclipse的更新步骤：Help >> Install New Software >> add，地址为http://download.eclipse.org/tools/gef/updates/interim/ ，直接按照提示安装就可以了，如果提示插件没有证书是否安装，选择ok就可以。
  • 安装m2e-wtp插件，这个插件的作用是将maven project 自动转为web project，方便直接部署到eclipse的tomcat中，更新地址为http://download.eclipse.org/m2e-wtp/snapshots/kepler

安装SVN插件和Check out as maven project插件

  • 安装subclipse 插件(即SVN插件)，更新地址 http://subclipse.tigris.org/update_1.8.x
  • 安装m2eclipse-subclipse，地址：http://repo1.maven.org/maven2/.m2e/connectors/m2eclipse-subclipse/0.13.0/N/0.13.0.201303011221

安装tomcat server 配置插件
　如果preferences中找不到server或者server中找不到tomcat的配置，请按以下步骤安装插件：

  • Help->InstallNew Software
  • 点击“add”按钮，输入
    name: “Kepler”
    repository:http://download.eclipse.org/releases/kepler
  • 找到“Web,XML, Java EE and OSGi Enterprise Development”——>“JST Server AdaptersExtensions”(打勾选上该项)，然后一路NEXT 完成安装。
